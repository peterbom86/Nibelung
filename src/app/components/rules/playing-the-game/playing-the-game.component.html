Playing Niebelung

Before the game can begin, the players must draft the scenario, select their (sub) faction and setup the battlefield. Here you will find the rules for doing exactly so.

Faction announcement

You begin the game by announcing your faction of choice to the opponent and show them your selection of scenario cards. A 'Scenario card' comes in four types, The 'Battlefield' cards, the 'Holding Objective' cards, the 'Extracting Objective' cards and the 'Denizens of the Land' cards. Each player should have selected 3 of each type of card for a total of 12 cards.

Scenario Selection, battlefield setup

Next is drafting the scenario cards; the players put all of the scenario cards on the table sorted according to their type. Then, the players roll of to see who is 'first-bidder' (denoted Player A from now on, his opponent is Player B). The 'first-bidder' is the player who gets to bid on a scenario card first. In order to bid on a scenario card, Player A puts one Command token on any scenario card. Player B then has a choice, either accept the bid or place a Command token himself.

Accepting the bid

If a player accepts a bid, he takes all Command tokens at the chosen Scenario Card and this card is now the selection for that type of card. Remove the remaining cards of the same type. Note that a player can only select a Scenario Card with his opponents tokens on it, not his own and not an empty card. This means that if all of the Scenario Cards of a certain type have none of his opponents tokens, then he cannot select that type of card.

Placing another Command Point

If the active player selects to place another Command token, he/she puts another Command Point on any remaining card that have none of his opponents Command tokens and pass the selection to his opponent.

At the end of the selection process, there should be four selected Scenario Cards, one of each type, and each player should have some amount of Command tokens. These tokens are kept during the game and can be spend similar to Command Points. The player with the most command tokens are said to have the initiative in the following steps.

The game is played on a 36" by 36" battlefield, now the table should be setup according to the rules described on the chosen Scenario cards.

Army Selection & Deployment

After the battlefield has been setup, You each (in secret) select your forces according to the engagement level determined by the 'Battlefield' card. Then, Going by initiative as determined earlier, you each deploy one unit at a time.

Game Start

Game is now ready to start, in the first round, initiative is already determined, so this can be skipped. The players now replenish resources and the first turn begins with the starting player taking an action.

Resolve rounds

Each of the turns are resolved with the players taking a turn - this goes on until both players have selected to pass, after which the magic, shooting, melee, scoring and cleanup phases are resolved. After this, the round has ended.

Resolving End of Round

At the end of each round, victory points are scored, game-end conditions are checked and if not, a new round is started

Game end

When one of the game-end conditions are triggered, the game ends and a winner is determined
