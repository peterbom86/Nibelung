Playing Niebelung

This is a short overview of how the game is played.

Game Setup

Setup the game board, determine starting armies and get ready to play.

Faction announcement

You begin the game by announcing your faction of choice to the opponent.

Scenario Selection, terrain placement

You select the scenario by choosing between a few different options for scenario, objectives and denizens - these also determine the terrain setup

Army Selection

You each (in secret) select your sub-faction which determines your starting force

Deployment

Going by initiative as determined earlier, you each deploy one unit at a time.

Game Start

Game is now ready to start, in the first round, initiative is already determined, so this can be skipped

Playing a Round

Resolve start of round, start

Resolve start of round

First round is started by resolving the 'Start of Round' sequence
At the start of each round, you replenish resources, pay upkeep and determine initiative for the first turn.

Resolve turns

Each of the turn phases are resolved with the players alternating between having the initiative

Resolve end of round

At the end of each round, victory points are scored, until-end-of-round effects are cleaned up, game-end is checked and if not, a new round is started

Game end

When one of the game-end conditions are triggered, the game ends and a winner is determined
