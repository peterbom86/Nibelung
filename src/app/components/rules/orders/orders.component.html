Orders is the way you command units to take action during a turn.

When giving out a shoot or charge order, turn the token facedown to remember that the unit will be shooting in the shooting phase or have charged for the melee phase.

Profile of an order

Stat Description
Order description What the order allows the unit to do
Order cost How many command points the order costs (you can only issue orders up to your command point total, command points are refreshed every turn)
