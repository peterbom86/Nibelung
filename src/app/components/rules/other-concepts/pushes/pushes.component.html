When pushing an enemy unit, you move the unit the pushed distance as if doing a normal movement with the unit

After pushing the enemy unit, you must place the pushing unit in base contact with the pushed unit, as close to its original position as possible

A pushed unit can only be pushed so far as to still be able to make contact with the pushing unit. This can shorten pushes if a larger based unit tries to push a smaller based unit in between impassible terrain or other units.

You can only push an enemy unit once per combat phase, further pushes has no effect
