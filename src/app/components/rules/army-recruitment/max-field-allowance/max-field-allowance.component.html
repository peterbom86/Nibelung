Max field allowance

Every army starts with some predefined starting units. From there on, you can, at the start of every round after the first, recruit additional troops for a cost in CP. When recruiting you must obey the following general unit limits unless army specific rules overrule this.

Unit type Max Allowance
Infantry max 3 of the same type
Mounted units max 2 of the same type
Flying units max 2 of the same type
Named characters max 1 of the same type
Warmachines max 2 of the same type
