A unit has a profile which consists of the following stats:

Stat Description
Movement / Charge Movement The max movement allowance when moving or charging
Attack - Range Melee or Ranged
Attack - Dice Number of dice rolled / number of dice picked
Attack - Attacks Numbers of attacks it can do each activation
Attack - Rend The amount of Resilience it can ignore when doing attacks
Attack - Charge Bonus The bonus dice given on the turn of a charge
Attack - Flank Bonus The bonus dice given when flanking an enemy unit
Defence - melee / shooting / magic Reduces the amount of hits by a set amount
Resilience The amount of damage subtracted whenever damage is dealt to the unit
Hit Points How many damage points each model can take before it is removed
Combat track The available options when a unit has reached a certain amount of hits against an enemy
Requstion points The requisition cost of the unit, used when recruiting the unit during the battle.
Leveling ability Units can gain new abilities by paying experience points, these each have a cost in experience points and an ability text.
Battlefield objective A description of the steps necessary for the unit to score VP by completing some action.
